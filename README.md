# Pseudocode Exercise 6 Array

---

## Exercise 1

Find average of list elements.

    Example:
    Input: [1,2,3,4,5] 
    Output: 3


## Exercise 2

Find smallest element in a list.

    Example:
    Input: [1,4,8,10,15]
    Output: 1

    
## Exercise 3

Find index number of highest number. Suppose index starts from 1.

    Example:
    Input: [1,5,9,3,4]
    Output: 3


## Exercise 4

Reverse the elements of a array in place.

    Example:
    Input: [1,2,3,4,5]
    Output: [5,4,3,2,1]
